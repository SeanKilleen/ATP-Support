Changes released Wednesday April 18, 2018 (USAJOBS Release 7.0)

This release of USAJOBS focused on performance, email infrastructure, hiring path, and SIF improvements.

We made a number of changes including:

Improved page load speed on a few critical pages

We revised the CSS and Javascript loaded in our application guide, profile dashboard, and landing pages to reduce the page load times. We also cached some database queries to further improve performance for job seekers.

Session inactivity decreased

For security we decreased the session inactivity from 30 to 15 minutes.

Clarification from the agency text wrapping

We fixed the text wrapping in the Clarification from the agency text.

Increased Education field character count limit

We increased the number of characters allowed in the Education field from 2,000 to 6,000 to handle interdisciplinary and other jobs with more extensive educational requirements.
